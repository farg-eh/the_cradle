# the_cradle
a 2D rpg game about stories from the cradle of civilization the lands of the tigris and the euphrates tigris and euphrates

> note : this game is part of my graduation project 

## overview 
the cradle is a 2D role playing game designed to put u inside some historical events so u can have fun while learning about history 
u can do quests, basic combat, mini games and choose different paths and get different endings 

## tech-stack
- python 3.13.5
- pygame-ce 2.5.5

## running the project
first u need python installed 
after that install pygame-ce with:
```
pip3 install pygame-ce
```
clone or downloaded this repo 
then u can run the game with:
```
python3 main.py
```
